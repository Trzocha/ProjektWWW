Witam!

Nazywam się Trzoska Sebastian. Jestem na początku swojej przygody z Front-Endem, którą rozwijam od 4 miesięcy. Przeszedłem kurs tworzenia Stron WWW w HTML i CSS i JS od Podstaw do Eksperta na platformie Udemy.

Z programowaniem mam do czynienia od 1.5 roku. Korzęnnm językiem jest C++ i na moim GitHub znajduję sie kilka zadań wraz z rozwiązaniem właśnie w tym języku.

Szukam pracy jako Junior Frond-End Developer aby robić to co lubię i spełniać sie zawodowo. Jestem gotowy na wyzwania oraz rozwój.

Aktualny projekt będzie się rozrastał w zależności od pomysłów bądż inspiracji jakie mną pokierują.
